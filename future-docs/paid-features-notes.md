# Binky Future Features + Paid Options

## Current Core Features (Shipping / Core Value)

- Organizer-first Downloads sorting into starter buckets (Images, Documents, Media, Archives, Apps, Screenshots, Misc, Review).
- Global watch folder with optional per-profile watch folders.
- Profiles with custom Finder tags, per-profile routing rules, "New" tag expiry, and optional post-sort Shortcuts action.
- Global output routing rules, exclusions, and dry-run preview.
- Sort summaries, undo flows, history, and menu bar controls.
- Keyboard shortcuts and menu bar only mode.

## Candidate Paid Features (Binky Plus Concepts)

- Multi-step routing automation:
  - Conditional rules with richer predicates (sender, source app, regex, file metadata bundles).
  - Rule templates and reusable rule packs.
  - Rule conflict simulator with "why this matched" diagnostics.

- Power profiles:
  - Profile stacks (fallback chains).
  - Schedule-aware profiles (workday/weekend contexts).
  - Team/profile export-import bundles.

- Smart classification upgrades:
  - Better screenshot grouping and semantic document clustering.
  - Duplicate/near-duplicate detection and merge suggestions.
  - AI naming suggestions for ambiguous files.

- Advanced workflows:
  - Multi-destination fan-out (copy vs move policies by rule).
  - Advanced post-sort actions (webhooks, AppleScript chains, Shortcuts parameter mapping).
  - "Review Inbox Assistant" with one-click bulk decisions.

- Insights and reporting:
  - Weekly cleanup reports.
  - Category trends and "top noisy sources" insights.
  - Saved-space/value metrics for organizer activity.

- Quality-of-life pro tools:
  - Rule set versioning + rollback.
  - Profile-level test sandbox with snapshot comparisons.
  - Smart onboarding wizard for large/messy Downloads folders.

## Packaging Thoughts

- Keep core sorting + one profile fully free.
- Paid tier should emphasize automation depth and confidence tools, not basic calm-inbox utility.
- Strong "Plus" value framing:
  - "Runs your file routing like a real assistant."
  - "More rules, more context, less manual cleanup."
